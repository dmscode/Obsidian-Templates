## 使用方法

首先请阅读：[Templater 脚本的使用方法](../Usages/How-to-Use-Templater-Script.md)

安装后在笔记中插入此模板便会添加题头图片，渐变色随机选用。文字内容为当前笔记的标题（文件名），添加后可以自行修改文字。
